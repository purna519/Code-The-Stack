CSS BOX Model
=================

WHAT IS THE BOX MODEL?
--------------------------

The CSS Box Model is a fundamental concept that defines how elements are structured and spaced on a web page. Every element is essentially a box composed of the following parts:

Content – The actual content (text, image, etc.) inside the box.
Padding – The space between the content and the border.
Border – The edge around the padding and content.
Margin – The space outside the element, separating it from other elements.

The order from inside out:
Content → Padding → Border → Margin

Example Visual Representation:

[ Border ]
[ Padding ]
[ Content ]

PARTS OF BOX MODEL
=====================

CONTENT
-----------
Defined using width and height properties.

This is where text, images, or other elements reside.
Example:
width: 250px;
height: 200px;

PADDING
--------
Space between content and border.

Pushes the border outward, increasing the element's box size (unless using box-sizing: border-box).
Shorthand:
padding: top right bottom left;
Examples:
padding: 10px;
padding: 10px 20px;
padding: 10px 15px 5px 0;

BORDER
----------
A line around the padding and content.

Can be styled using color, width, and style.
Shorthand:
border: width style color;
Examples:
border: 2px solid black;
border: 1px dashed red;

MARGIN
----------
Space between elements (outside the border).

Used to separate elements from each other.
Shorthand:
margin: top right bottom left;
Examples:
margin: 10px;
margin: 5px 10px;
margin: 0 auto; (to center an element horizontally)

box-sizing PROPERTY
---------------------
Controls how total width and height are calculated.

Default behavior:
box-sizing: content-box;

Width and height include only the content.
Padding and border are added outside this size.
Recommended behavior:
box-sizing: border-box;
Width and height include content, padding, and border.

Best practice:
----------------
{
box-sizing: border-box;
}

REAL-WORLD EXAMPLE
---------------------

.profileCard {
width: 250px; (Content width)
padding: 10px; (Padding inside the element)
border-radius: 5px; (Rounded corners)
margin: 10px; (Spacing between cards)
box-shadow: ...; (Shadow outside the box)
}

.profileCard img {
border: 5px solid white; (Image border)
}

BOX MODEL TIP
---------------
Use browser developer tools to inspect the box model:
Right-click an element → Inspect → Check the computed layout box (usually in right sidebar). It visually shows margin, border, padding, and content.

COMMON ISSUES AND SOLUTIONS
-----------------------------
Issue: Element appears bigger than expected.
Fix: Use box-sizing: border-box to include padding and border in total width/height.

Issue: Elements too close to each other.
Fix: Add margin for spacing.

Issue: Text too close to the border.
Fix: Add padding.

SUMMARY
----------
Content is controlled with width and height.
Padding adds space inside the element.
Border surrounds the content and padding.
Margin adds space outside the element.
Use box-sizing: border-box for accurate layout control.
