Anchor Tag and Layout
===================================================================================================

Anchor Tag Basics

The <a> tag is used to create clickable links.
Links can point to webpages, sections within a page, emails, or files.
Syntax: <a href="URL">Link Text</a>

===================================================================================================

Using div for Layout

<div> is a block-level container used to group and organize HTML elements.
Helps in structuring the webpage visually.
You can use <div>s to arrange links in rows, columns, menus, or sections.
With CSS, <div>s can be styled to control positioning, spacing, and appearance.

===================================================================================================

Why Use div with Anchor Tags?

To create navigation bars or menus.
To group related links.
To apply styles (like background, padding, margins) to a group of links.
To control layout using CSS (flexbox, grid, floats, etc.)

===================================================================================================

Common Layout Patterns

Horizontal navigation menu: Use a div with inline or flex display for links side-by-side.
Vertical link list: Use a div with block-level links stacked vertically.
Button-like links: Wrap links in divs or style them to look like buttons.

===================================================================================================

CSS for Layout

Use display: flex or display: inline-block on div or links for horizontal alignment.
Use margins and padding to space links.
Use text-align or flex properties to control alignment.
